## Image size guide

| Location          | Orientation | Recommended size  |
|-------------------|-------------|-------------------|
| hero.jpg          | Landscape   | 1800 × 700 px     |
| portrait-N.jpg    | Portrait    | 600 × 800 px      |
| gallery-N.jpg     | Mixed       | 900+ px longest   |
| covers/           | Landscape   | 900 × 675 px      |
| photographer.jpg  | Portrait    | 800 × 1067 px     |
| qr-code.png       | Square      | 400 × 400 px      |

## Personalising contact info

Open each file and find `YOUR-EMAIL-HERE`, `YOUR-PHONE-HERE`, and the Instagram `href="#"` — replace with your real details. These appear in: index.html, about.html, galleries.html, contact.html.

---

## Adding a new gallery later

1. Upload a cover image to `images/covers/yournewgallery-cover.jpg`
2. Open `galleries.html` and copy one of the `<div class="gallery-card">` blocks
3. Update the image path, title, and series number
4. Remove the "Coming Soon" badge line
5. Create a new page `galleries/yournewgallery.html` for the photos
